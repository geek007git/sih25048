What is Algebra?

Algebra is the branch of mathematics that uses variables (letters) to represent unknown numbers and establishes relationships through equations. It's the language we use to describe patterns, solve problems, and model real-world situations where some quantities are unknown.

Variables and Expressions

Variables are symbols (usually letters like x, y, or z) that represent unknown values. An expression combines variables, numbers, and operations to describe a mathematical relationship. Unlike equations, expressions don't have an equals sign.

3x + 5y - 2
This is an algebraic expression where:
3x = 3 multiplied by variable x
5y = 5 multiplied by variable y
-2 = a constant term
The expression represents different values depending on x and y

Solving Linear Equations

Equations are mathematical statements that two expressions are equal, connected by an equals sign. Solving an equation means finding the value of the variable that makes the equation true. The key principle is maintaining balance - whatever you do to one side, you must do to the other.

2x + 3 = 11
Step-by-step solution:
2x + 3 = 11 (original equation)
2x = 11 - 3 (subtract 3 from both sides)
2x = 8 (simplify)
x = 4 (divide both sides by 2)

Key Concepts

Variables

Letters that represent unknown numbers. They allow us to write general rules and solve problems where quantities are unknown or changing.

Equations

Mathematical statements showing that two expressions are equal. The foundation for problem-solving in algebra and beyond.

Operations

Addition, subtraction, multiplication, and division work the same with variables as with numbers, following the same mathematical rules.
